# PickMe Nails  — Telegram-бot для записи на маникюр

## Структура проекта

```
nail_bot/
├── .env.example          # шаблон переменных окружения
├── requirements.txt      # зависимости
├── config.py             # загрузка настроек (pydantic-settings)
├── main.py               # точка входа
├── database/
│   ├── models.py          # SQLAlchemy-модели (User, Service, Booking)
│   ├── db_main.py         # engine, sessionmaker, init_db()
│   └── requests.py        # CRUD-функции
├── states/
│   ├── booking.py          # FSM клиента
│   └── admin.py            # FSM администратора
├── keyboards/
│   ├── client_kb.py         # клавиатуры клиента
│   └── admin_kb.py          # клавиатуры админа
└── handlers/
    ├── common.py            # /start, /cancel
    ├── client.py            # запись, мои записи, цены, контакты
    └── admin.py              # /admin, управление услугами и записями
```


## Основные команды

- `/start` — главное меню
- `/cancel` — сброс текущего диалога (FSM)
- `/admin` — панель администратора (доступна только ID из `ADMIN_IDS`)

## Логика записи

1. Клиент выбирает услугу → дату (ближайшие 7 дней) → свободное время (10:00, 13:00, 16:00, 19:00).
2. Если номер телефона ещё не сохранён — бот запрашивает его через кнопку «Отправить контакт» либо текстом.
3. Клиент видит итоговую сводку и подтверждает запись.
4. Запись сохраняется в БД со статусом `confirmed`, администраторам приходит уведомление.
5. В «Мои записи» клиент может отменить свою запись — админ получит уведомление об отмене.
6. Админ может посмотреть все записи на сегодня и отменить любую из них — клиенту придёт уведомление.
